## Accounting ERP APIs

Tools for integrating with Swedish accounting and ERP systems.

- **[Fortnox API](https://developer.fortnox.se/)** — REST API for Sweden's most used accounting platform. Covers invoices, vouchers, accounts, customers and more. Used by 450,000+ companies.
- **[Visma eAccounting API](https://developer.visma.com/)** — API for Visma's cloud accounting suite. Supports invoices, suppliers, and financial data.
- **[Bjorn Lunden API](https://www.bjornlunden.se/program/api)** — Integration API for BL Bokföring and BL Administration.
- **[Spiris API](https://www.spiris.se/)** (formerly Visma Spcs) — API access to Spiris accounting products including Visma Administration.
- **[Hogia API](https://www.hogia.se/)** — APIs for Hogia's payroll and accounting suite, widely used in Swedish municipalities and enterprises.
- **[PE Accounting API](https://www.peaccounting.com/)** — Modern accounting platform with API, popular with consulting firms.

## Payment & Banking

- **[Stripe](https://stripe.com/se)** — Full payment infrastructure with SEK support, Klarna integration, and Swedish VAT handling.
- **[Klarna API](https://developers.klarna.com/)** — Buy now, pay later + payments. The most recognized Swedish fintech brand globally.
- **[Swish API](https://www.swish.nu/developer)** — Swedish mobile payment standard. Used by 8.5M+ Swedes. Required for consumer-facing apps.
- **[Bankgiro API](https://www.bankgirot.se/)** — Bankgiro payment processing, standard for B2B payments in Sweden.
- **[Autogiro](https://www.bankgirot.se/vara-tjanster/ta-betalt/autogiro/)** — Direct debit standard for recurring payments in Sweden.
- **[Nets Easy](https://developers.nets.eu/)** — Nordic payment gateway with strong Sweden coverage.

## Identity & Authentication

- **[BankID](https://www.bankid.com/en/utvecklare)** — Sweden's national digital identity standard. Required for legally binding signatures and age verification.
- **[Freja eID](https://frejaeid.com/developers/)** — Alternative to BankID, simpler integration for some use cases.
- **[SITHS](https://inera.se/e-halsa/siths/)** — Smart card-based identity for Swedish healthcare and government systems.

## Tax & Regulatory

- **[Skatteverket API](https://skatteverket.se/omoss/apierochoppnadata.4.html)** — Official Swedish Tax Agency APIs. Covers VAT registration lookup, F-tax verification, and more.
- **[Bolagsverket API](https://www.bolagsverket.se/)** — Company registration and corporate data from the Swedish Companies Registration Office.
- **[allabolag.se](https://www.allabolag.se/)** — Comprehensive Swedish company data including financials (no official API, scraping possible).
- **[UC API](https://www.uc.se/foretag/api/)** — Credit scoring and business intelligence for Swedish companies.
- **[Creditsafe](https://www.creditsafe.com/se/sv/api.html)** — International credit data with strong Swedish coverage.

## Open Banking

- **[Tink](https://tink.com/developers/)** — Europe's leading open banking platform, founded in Stockholm. PSD2 account aggregation + payment initiation for all Swedish banks.
- **[Aiia](https://aiia.eu/)** — Nordic open banking API, strong coverage of Scandinavian banks.
- **[Nordea Open Banking](https://developer.nordeaopenbanking.com/)** — Direct API access to Nordea accounts (Sweden's largest bank).
- **[SEB Open Banking](https://developer.seb.se/)** — SEB's PSD2 API for account information and payment initiation.
- **[Swedbank Open Banking](https://developer.swedbank.com/)** — Swedbank and Sparbanken APIs.
- **[Handelsbanken API](https://developer.handelsbanken.com/)** — Open banking API for Handelsbanken accounts.

## Libraries & SDKs

- **[fortnox-node](https://www.npmjs.com/package/fortnox-node)** — Unofficial Node.js client for the Fortnox API.
- **[bankid](https://www.npmjs.com/package/bankid)** — Node.js library for BankID integration.
- **[swish-merchant](https://www.npmjs.com/package/swish-merchant)** — Node.js Swish merchant API client.
- **[personnummer](https://www.npmjs.com/package/personnummer)** — Swedish personal identity number (personnummer) validation in JavaScript.
- **[org-nr](https://www.npmjs.com/package/org-nr)** — Swedish organization number validation.
